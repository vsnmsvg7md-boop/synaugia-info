Privacy Policy — SYNAUGIA

Last updated: April 26, 2026

SYNAUGIA (“the App”) is an intelligent harmony and improvisation tool designed for musicians. This Privacy Policy explains how information is handled when you use the App.

1. Data Collection

SYNAUGIA does not require account creation and does not intentionally collect personally identifiable information such as name, email address, or location. However, the App processes user input data (such as chords, musical sequences, or queries) in order to generate harmonic analysis and suggestions.

2. Use of AI Services

To provide its core functionality, SYNAUGIA sends user input to an external AI service via a secure proxy server.

  • The input data is used only to generate responses.
  • No user profiles are created.
  • No personal tracking is performed.

3. Data Storage

SYNAUGIA does not store user inputs or generated results on its servers.

  • No database of user activity is maintained.
  • No long-term storage of musical queries.

4. Analytics and Tracking

SYNAUGIA does not use third-party analytics tools, advertising trackers, or cookies.

5. Security

All communication between the App and external services is performed via secure connections (HTTPS). An intermediary proxy server is used to protect API credentials and prevent unauthorized access.
