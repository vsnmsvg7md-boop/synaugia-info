Privacy Policy — SYNAUGIA

Last updated: 04/26/2026

SYNAUGIA (“the App”) is an intelligent harmony and improvisation tool designed for musicians[cite: 1]. This Privacy Policy explains how information is handled when you use the App[cite: 2].

1. Data Collection

SYNAUGIA does not require account creation and does not intentionally collect personally identifiable information such as name, email address, or location[cite: 3]. However, the App processes user input data (such as chords, musical sequences, or queries) in order to generate harmonic analysis and suggestions[cite: 4].

2. Use of AI Services

To provide its core functionality, SYNAUGIA sends user input to an external AI service via a secure proxy server[cite: 5].

  • The input data is used only to generate responses[cite: 6].
  • No user profiles are created[cite: 6].
  • No personal tracking is performed[cite: 6].

The AI provider may process requests temporarily to generate results, in accordance with their own privacy policies[cite: 6].

3. Data Storage

SYNAUGIA does not store user inputs or generated results on its servers[cite: 7].

  • No database of user activity is maintained[cite: 8].
  • No long-term storage of musical queries[cite: 8].

Temporary processing may occur during API requests, but data is not retained after the response is delivered[cite: 8].

4. Analytics and Tracking

SYNAUGIA does not use third-party analytics tools, advertising trackers, or cookies[cite: 9].

5. Security

All communication between the App and external services is performed via secure connections (HTTPS)[cite: 10]. An intermediary proxy server is used to protect API credentials and prevent unauthorized access[cite: 11].

6. Third-Party Services

SYNAUGIA relies on third-party AI infrastructure to generate musical analysis. These services may process input data transiently[cite: 12]. Users are encouraged to review the privacy policies of these providers[cite: 13].

7. Children’s Privacy

SYNAUGIA is not specifically directed toward children under the age of 13. The App does not knowingly collect personal data from children[cite: 14].

8. Changes to This Policy

This Privacy Policy may be updated from time to time[cite: 15]. Any changes will be reflected on this page with an updated date[cite: 16].

Summary

SYNAUGIA is designed to respect user privacy:

  • No accounts
  • No personal data collection
  • No tracking
  • No data storage

Only the information necessary to generate musical analysis is processed temporarily.
